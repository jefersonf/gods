# Go Data Structures

Go (uncommon) data structures

## Requirements

- Go 1.18+

## Implementations

### Fundamentals

- Queue
- Stack

### Tree

- DSU (Disjoint-Set Union)
- Segment Tree
- Fenwick (Binary-index Tree)
- Trie

## Tests

$ cd tree/segment
